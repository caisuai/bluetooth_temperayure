基于FreeRTOS的室内湿度智能调节系统，以STM32F103为主控芯片，实现环境温湿度实时监测、自动/手动加湿、OLED显示及蓝牙远程反馈。

系统架构：在FreeRTOS环境下创建多个独立任务，包括数据采集任务、显示任务、蓝牙通信任务，数据处理任务，通过消息队列和信号量实现任务间数据同步与互斥。

传感器与执行：DHT20温湿度传感器定期采集数据；通过继电器模块控制加湿器/除湿器，设定湿度阈值自动启动，同时支持手动强制开关。

人机交互：采用旋转编码器与独立按键进行阈值调节，并通过0.96英寸OLED（I2C）实时显示当前温湿度、设备状态及可调阈值。

远程通信：通过UART连接蓝牙模块，将温湿度数据、设备状态及报警信息实时发送至手机APP。

开发环境：基于STM32CubeMX生成初始化代码，使用Keil MDK编写C语言程序，结合FreeRTOS内核进行任务调度与资源管理。
